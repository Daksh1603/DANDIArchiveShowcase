# Summary statistics for the available dandisets (Updated on 2023-08-27)



## NWB dandisets

- Total number of NWB dandisets: 9

- Median number of files in each NWB dandiset: 169.0

- Median number of bytes in each NWB dandiset: 55,585,858,956

- 6 most commonly measured variables: ProcessingModule, BehavioralTimeSeries, Units, BehavioralEvents, PlaneSegmentation, ElectrodeGroup

- NWB dandisets that pass NWBInspector and thus are possibly NWBE compatible

[000013](https://dandiarchive.org/dandiset/000013), [000015](https://dandiarchive.org/dandiset/000015), [000019](https://dandiarchive.org/dandiset/000019), [000021](https://dandiarchive.org/dandiset/000021)

- NWBE compatibility terminology: 
  - LL-P: Likely plottable - file whose datatypes extend TimeSeries that can be viewed and plotted 
  - LL-V: Likely viewable - file whose datatypes might not extend TimeSeries that can be viewed 
  - NC-0: Not compatible level 0 - file cannot be opened 
  - NC-1: Not compatible level 1 - geppetto model for file cannot be created 
  - NI: No information - file is not tested
